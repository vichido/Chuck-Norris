Readme
======

1. Install cocoa pods:
``gem install cocoapods \n
pod setup``         


2. Clone ``https://github.com/johncadengo/Chuck-Norris.git``

3. Create a Podfile where the ``Chuck.xcodeproj`` file is.
	``platform :ios, '5.0' \n
	pod 'RestKit', '~> 0.20.0' `` 

4. Run ``pod install``

5. Open the file Chuck.xcworkspace and run
